SYSTEM REQUIREMENTS:
--------------------
- Node.js (v14 or higher)
- npm (Node Package Manager)
- Modern web browser (Chrome, Firefox, Safari, Edge)

INSTALLATION STEPS:
-------------------
1. Download and extract the project folder

2. Ensure the following folder structure is maintained:

   project-folder/
   ├── server.js
   ├── package.json
   ├── database.sqlite (will be auto-generated)
   ├── index.html
   ├── css/
   │   └── (all CSS files)
   └── pages-html/
       └── (all HTML files)

3. Install dependencies:
   Open terminal in project folder and run:
   > npm install

4. Start the server:
   > node server.js

5. Open your browser and navigate to:
   > http://localhost:3000

TROUBLESHOOTING:
----------------
- If server fails to start: Check if port 3000 is already in use
- If database error occurs: Delete database.sqlite and restart server
- If styles don't load: Check that all CSS files are in the css/ folder
- For best experience: Use Chrome or Firefox
